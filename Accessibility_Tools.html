Accessibility  Tools

Read Aloud

Microsoft Edge's Read Aloud feature is a built-in browser tool that can enhance accessibility on an LMS. This text-to-speech tool reads web pages, PDFs, and documents aloud in the browser, making content more accessible to students with visual impairments or reading difficulties.

Starting the narration is easy with shortcuts, and the tool also offers customization options such as adjusting the reading speed and choosing from various voices and accents, allowing users to tailor the reading experience to their preferences.

Test Responsive

Test Responsive is designed to help developers and educators test web designs and mobile pages across various devices. It allows users to view websites in multiple screen modes, emulating different user agents and screen resolutions. This tool is particularly useful for ensuring that educational content is accessible and visually consistent across different devices, such as smartphones and tablets.

Educators and EdTechs can use Test Responsive to verify that online learning materials, such as course websites and digital textbooks, are properly displayed on various devices. This ensures that all students, regardless of the device they use, have a seamless and effective learning experience. By testing the responsiveness of educational websites, educators can identify and fix any display issues that might hinder student engagement or comprehension.

Test Responsive as a tool promotes inclusivity and accessibility. It helps institutions ensure that their digital resources are optimized for all students, enhancing the overall quality of online education. This tool also supports the development of mobile-friendly educational content, which is increasingly important as more students access learning materials on their mobile devices.
